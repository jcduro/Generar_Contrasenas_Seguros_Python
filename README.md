# 🔒 Generador de Contraseñas Seguras con Python y Tkinter

---

📘 Descripción

Este generador de contraseñas te permite crear claves seguras personalizadas desde una interfaz moderna y sencilla, utilizando Python y Tkinter.  
Puedes elegir el largo y los tipos de caracteres (mayúsculas, minúsculas, números y símbolos). ¡Ideal para reforzar la seguridad de tus cuentas!

---

🧰 Tecnologías Utilizadas

🐍 Python 3.x

## 📊  Características

- Interfaz gráfica intuitiva y moderna (con checkboxes claros)
- Permite seleccionar una o varias opciones de caracteres:
  - Mayúsculas (A-Z)
  - Minúsculas (a-z)
  - Números (0-9)
  - Símbolos (# $ % & ...)
- Elige longitud de la contraseña fácilmente (por defecto: 12)
- La contraseña generada se copia automáticamente al portapapeles
- Mensajes claros si falta seleccionar opciones o el largo es inválido
- 100% personalizable y ejecutable en Windows, Linux o MacOS

---

## Instalación y Uso

1. **Instala Python 3.x:**  
   Descárgalo desde [python.org](https://www.python.org/).

2. **Guarda el script:**  
   Descarga el archivo `password_generator.py` en tu PC.

3. **Ejecuta el script:**  
   - Abre la terminal o CMD (puedes usar Visual Studio Code).
   - Ve a la carpeta del script y ejecuta:
     ```
     python password_generator.py
     ```

4. **Usa la interfaz:**  
   - Ingresa la longitud que deseas.
   - Marca las casillas para elegir mayúsculas, minúsculas, números y/o símbolos.
   - Haz clic en "Generar Contraseña".
   - ¡La clave segura se mostrará y se copiará al portapapeles automáticamente!
